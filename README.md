# 箱子終端
黏液科技的附加 繁體翻譯版

此附加靈感來源於著名的模組[Applied Energistics](https://www.curseforge.com/minecraft/mc-mods/applied-energistics-2).<br>
它可添加一個箱子終端讓所有連結物流系統上節點設置為CT-Channel可訪問所有物品.

[官方原碼連結](https://github.com/TheBusyBiscuit/ChestTerminal)
